# Mostakil AI — archived

> **This repo has been superseded by [`groupsmix/mostaqil-ai`](https://github.com/groupsmix/mostaqil-ai).**
>
> `mostakil` is a non-standard romanization of مستقل; the canonical repo uses the more standard transliteration `mostaqil`. All future work on the Arabic AI-freelancing ebook launch funnel — landing page, FastAPI lead-capture backend, and the 90-page ebook build pipeline — lives there.

## What happened

This repo originally held an Astro 6 + Tailwind 4 static landing page for the **مستقل الذكاء الاصطناعي** ebook. The long-form sales-page structure (Hero → Problem → Truth → What's Inside → Differentiators → Who For → Bonuses → Pricing → Guarantee → Testimonials → Lead Magnet → FAQ → Final CTA) has been merged into `groupsmix/mostaqil-ai` as its new `frontend/` alongside the pre-existing FastAPI backend and ebook pipeline.

See the merge PR: https://github.com/groupsmix/mostaqil-ai/pull/1
